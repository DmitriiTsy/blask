# Веб-интерфейс Blask

## 🚀 Быстрый запуск

### Вариант 1: Через скрипт (рекомендуется)
```bash
./run_webapp.sh
```

### Вариант 2: Прямой запуск
```bash
streamlit run src/webapp.py
```

### Вариант 3: С виртуальным окружением
```bash
source venv/bin/activate
streamlit run src/webapp.py
```

## 📋 Что нужно

1. **Установить Streamlit** (если еще не установлен):
```bash
pip install streamlit
```

2. **Настроить .env** с API ключами (см. ENV_VARIABLES.md)

3. **Запустить веб-приложение**

## 🌐 Использование

После запуска откроется веб-интерфейс по адресу:
- **http://localhost:8501**

### Возможности интерфейса:

1. **Ввод запроса** - текстовое поле для вашего запроса
2. **Примеры запросов** - готовые примеры в боковой панели
3. **Результаты**:
   - Решение системы (decision)
   - Тип поиска
   - Форматированный ответ
   - Визуализации (графики)
   - Результаты поиска
   - Debug информация

### Примеры использования:

- "What are the latest AI trends?"
- "Who are competitors in e-commerce?"
- "Show Python usage statistics for the last year"
- "What's trending in machine learning?"

## 🎨 Особенности

- **Простой и интуитивный интерфейс**
- **Примеры запросов** в боковой панели
- **Автоматическое отображение графиков**
- **Просмотр результатов поиска**
- **Debug режим** для разработчиков

## 🔧 Настройка

### Изменить порт:
```bash
streamlit run src/webapp.py --server.port 8502
```

### Запустить в headless режиме:
```bash
streamlit run src/webapp.py --server.headless true
```

## 📱 Доступ

После запуска веб-интерфейс доступен:
- Локально: http://localhost:8501
- В сети: http://your-ip:8501 (если настроено)

## 🛠️ Разработка

Код веб-приложения находится в `src/webapp.py`.

Основные компоненты:
- Streamlit для UI
- Интеграция с основным графом
- Отображение результатов и визуализаций
- Обработка ошибок
