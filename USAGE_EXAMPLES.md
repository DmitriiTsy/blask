# Примеры использования

## Базовое использование

### Интерактивный режим
```bash
python3 -m src.main
```
Затем введите ваш запрос когда появится приглашение.

### С аргументом
```bash
python3 -m src.main "Что в тренде в области AI?"
```

## Примеры запросов

### Поиск трендов
```bash
python3 -m src.main "Какие тренды в машинном обучении в 2024?"
```

### Анализ конкурентов
```bash
python3 -m src.main "Кто мои конкуренты в области e-commerce платформ?"
```

### Статистика с графиками
```bash
python3 -m src.main "Покажи статистику роста популярности Python за последние 5 лет"
```

### Поиск по ключевым словам
```bash
python3 -m src.main "Найди информацию о LangGraph и его возможностях"
```

### Комплексный запрос
```bash
python3 -m src.main "Проанализируй конкурентов в сфере AI-ассистентов и покажи тренды"
```

## Через скрипт

```bash
# Интерактивный режим
./run.sh

# С запросом
./run.sh "Ваш запрос здесь"
```

## Многословные запросы

Все аргументы после команды объединяются в один запрос:

```bash
python3 -m src.main Покажи мне статистику использования Python за последний год и создай график
```

## Что происходит

1. **Thinking Node** анализирует ваш запрос
2. Определяет тип (поиск, статистика, тренды)
3. **Search Node** выполняет поиск (если нужен)
4. **Analysis Node** обрабатывает и форматирует результат
5. Выводит ответ в консоль
