# Как запустить проект

## Быстрый запуск

### Вариант 1: Прямой запуск
```bash
python3 -m src.main
```

### Вариант 2: Через Python модуль
```bash
cd /Users/dmitrii/maincard/Blask
python3 -m src.main
```

## С виртуальным окружением (рекомендуется)

### 1. Активировать виртуальное окружение
```bash
source venv/bin/activate
```

### 2. Запустить проект
```bash
python -m src.main
```

## Настройка окружения

Перед запуском создайте файл `.env` с API ключами:

```bash
# Скопировать пример
cp .env.example .env

# Отредактировать .env и добавить ключи
nano .env
```

Минимально необходимые ключи:
```env
OPENAI_API_KEY=your_key_here
# или
ANTHROPIC_API_KEY=your_key_here
```

## Пример использования

После запуска проект выполнит пример запроса:
- "What are the latest AI trends?"
- Обработает через граф
- Выведет результат

## Запуск с кастомным запросом

### Вариант 1: Через аргументы командной строки
```bash
python3 -m src.main "Какие тренды в AI сейчас?"
```

Или через скрипт:
```bash
./run.sh "Ваш запрос здесь"
```

### Вариант 2: Интерактивный ввод
```bash
python3 -m src.main
# Затем введите ваш запрос когда появится приглашение
```

### Вариант 3: Многословный запрос
```bash
python3 -m src.main "Покажи мне статистику использования Python за последний год и создай график"
```

## Структура выполнения

1. **Thinking Node** - анализирует запрос
2. **Search Node** - выполняет поиск (если нужен)
3. **Analysis Node** - обрабатывает и форматирует результат

## Troubleshooting

### Ошибка: "No API key found"
- Создайте `.env` файл
- Добавьте `OPENAI_API_KEY` или `ANTHROPIC_API_KEY`

### Ошибка: "Module not found"
- Установите зависимости: `pip install -r requirements.txt`
- Или используйте виртуальное окружение

### Ошибка: "No module named src"
- Убедитесь, что вы в корневой директории проекта
- Запускайте: `python3 -m src.main` (не `python3 src/main.py`)
